SPORT SPORT Introduction SPORT is text-based offline software package which makes it possible to write ABB robot programs in the language Arla on an external personal computer and then download them to an robot controller. It is fully compatible with Arla programmes created by Workspace and may be used as an alternative to using ABB software to download these text files to an controller. The latest release of SPORT has the following capabilities File compilation Convert robot program from ARLA to binary code. Task compilation Convert all robot programs in task from ARLA to binary code. Diskette Functions Using diskette to transfer program between robot and PC. On Line Communication Communication between robot and PC via serial link. File decompilation Convert robot program from binary code to ARLA robot language. Block decompilation Convert block of robot programs from binary code to ARLA. Program Editor Creating robot programs and transfer position and orientation data via DDE to SPORT from AutoCAD. OLP-import file with .irb extension created in OLP english or swedish can be converted to SPORT standard. Printing Print file File Print notepad starts with the actual file you may add comments to the file Arkiv Skriv ut. Possible to specify Word processor WinWord Write Notepad... The word processor is started from the menu or by Ctrl Enter in .rpl-file or selected .rpl-file in the task-file. If file excedes kb the specified word processor will be started automatically. As complement to task new feature with block files extension .IRB has been introduced which is compatible with olp irb files. This version of SPORT has debugging capabilities. This application shows the current running line in the task by highlighting it. Prodebug.exe has DDE Dynamic Data Exchange to the to sailserv.exe module for robot control. This version is shipped with the ADS aplication SAIL Simulation AId for Link mechanisms This is general module for kinematic simulation in AutoCAD for Windows. Via DDE are the module SAILSERV and SAIL communicating and the model of the ABB are automatically updated during the simulation. New Features in the latest release Release adds new features to the SPORT system. TCP calibration You are now able to measure the TCP and Basepoint of the robot and directlly update the TCP register of the controller.

How to Obtain SIR-C Software How to Obtain SIR-C Software If you wish to obtain an mm tape with the SIR-C CEOS_READER software the data decompression software and the SIR-C/AIRSAR conversion software these are the steps that have to be taken print out the software licenses fill out and sign fax all portions to Attention Ellen Leary Radar Data Center Jet Propulsion Laboratory An mm tape containing the software with manuals will be mailed to you.
